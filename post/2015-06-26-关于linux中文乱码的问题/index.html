关于linux中文乱码的问题。

2015年06月26日

[TOC] 博客园文章地址 http://www.cnblogs.com/oloroso/archive/2015/06/26/4602697.html

公司让人在一台装有ubuntu14.04的机器上安装net-snmp，可是这台机器的设置很让人不喜。没关系，一个个解决它。

  • 不能连接外网，得弄一个代理。
    这个好说，在可以上外网的本机上安装squid工具来做代理即可。
  • locale看看，这个设置得改啊。

fx@fx:~$ locale
LANG=zh_CN.UTF-8
LANGUAGE=zh_CN:zh
LC_CTYPE=“zh_CN.UTF-8”
LC_NUMERIC=zh_CN
LC_TIME=zh_CN
LC_COLLATE=“zh_CN.UTF-8”
LC_MONETARY=zh_CN
LC_MESSAGES=“zh_CN.UTF-8”
LC_PAPER=zh_CN
LC_NAME=zh_CN
LC_ADDRESS=zh_CN
LC_TELEPHONE=zh_CN
LC_MEASUREMENT=zh_CN
LC_IDENTIFICATION=zh_CN
LC_ALL=
  • 没有安装中文语言包，先安装一个。
sudo apt-get install language-pack-zh-hans-base language-pack-zh-hans
